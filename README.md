## Projects

### 🔷 DocVault *(WIP — MCA Final Year Project)*
> Multi-tenant SaaS document management platform

`React.js` `Node.js` `Express` `PostgreSQL` `Prisma ORM` `AWS S3` `Redis` `Clerk` `Razorpay`

Designing a secure multi-tenant SaaS platform for organisations to store, share, version-control, and manage documents in isolated environments on shared cloud infrastructure.

- Researching a **direct-upload flow** using pre-signed S3 URLs so file data never touches the application server — encrypted in-transit via TLS and at rest via AWS SSE-S3, with metadata tracked in PostgreSQL.
- Planning full document lifecycle: **RBAC** across Owner, Admin, Editor, and Viewer tiers; S3 versioning for non-destructive updates; automated Glacier archival via lifecycle rules; soft delete with restore window.
